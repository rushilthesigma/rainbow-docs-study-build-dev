# RushilAI
Your new favorite AI EdTech assistant - seamlessly write curricula for your classes, learn new skills, and practice for debate and Quiz Bowl... all in one app.

## Features
- Notes built-in
- Curriculum Generator - for academic topics and life skills
- Versatile AI use - pick which model you want to use for each task
- Quiz-Bowl and Debate - Test your skills against friends!
